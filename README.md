# Desafio (Teste Automatizado Web)
É um projeto para automatizar cenário de compra WEB utilizando Java, Junit e Selenium 

## Pré-requisito
Baixar as bibliotecas JUNIT e SELENIUM WEBDRIVER
Baixar GECKODRIVER

## JUNIT
Baixe e inclua a biblioteca do JUNIT ao projeto - <a href="https://github.com/junit-team/junit4/wiki/Download-and-Install">Junit</a>

## Selenium
Baixe e inclua a bibliotecas do Selenium WEBDriver - <a href="https://www.selenium.dev/downloads/">Selenium Webdriver</a>

## Driver Firefox
Baixe o driver geckodriver e guarde em um diretório da sua preferência - <a href="https://github.com/mozilla/geckodriver/releases">Geckodriver</a>

## Variáveis de ambiente Windows

 - Acesse as variáveis de ambiente do sistema<br>
 - Em path inclua o diretorio onde está o geckodriver<br>
 OBS: Se não configurar a variavel de ambiente o driver não será encontrado
 
## Observação
Antes de executar o teste alterar as entradas conforme necessário no Teste.java
